Dartmouth Focus App Documentation

Welcome to the comprehensive documentation for the Dartmouth Focus App. This guide provides detailed information about the app's features, installation process, development guidelines, and machine learning capabilities.

Installation

Prerequisites

  • Xcode 14.0 or later
  • iOS 15.0+ deployment target
  • CocoaPods package manager
  • Git version control

Setup Steps

  1. Clone the repositories: 
    git clone [repository-url] Earable-iOS
git clone [repository-url] Earable-iOS-SDK
  2. Install dependencies: 
    cd Earable-iOS && pod install
cd ../Earable-iOS-SDK && pod install
  3. Open the workspace: 
    open Earable-iOS/EarableMobileLibrary.xcworkspace

Project Structure

Main Application (Earable-iOS)

  • ViewControllers/

    • FocusDashboardView: Main dashboard

SDK Implementation (Earable-iOS-SDK)

  • EarableMobileLibrary/

    • Signal processing
